Playwright API for Automated Browser Interaction.

1. Download Python

2. Create a virtual enviornment

python3 -m venv venv

 . venv/bin/activate

3. Install requirements.txt

pip install -r requirements.txt

4. Install Playwright

playwright install

4. Run the API:
uvicorn main:app
